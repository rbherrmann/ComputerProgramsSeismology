RFTN - Shallow Sediment Effect

Introduction

The objective of this example is to examine the effect of shallow low velocity sediments on the receiver function. A halfspace and a simple crustal model are considered.

Download

Download the package RFTN3.tgz. Then do the following: 

gunzip -c RFTN3.tgz | tar xvf -
cd RFTN3
./DOIT

The script will create the figures shown below. The script requires CPS programs hrftn96, gsac and plotnps and the ImageMagick package program convert to transform the EPS files to a PNG image.
