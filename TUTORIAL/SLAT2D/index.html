Computer Programs in Seismology Tutorial

Surface waves in 2-D media

Introduction

This is a short introduction to the use of the program slat2d96. This implements the work of Keilis-Borok and others so model fundamental mode surface waves as they propagate through laterally varying medium. This only works for the fundamental mode, and assumes conservation of energy.

The tutorial consists of a pdf file gChap4.pdf which describes the method. There is also gzip'd tar file that does the computations shown in the pdf file (ocean-continent.tgz). Unpack the archive using the commands: 

      gunzip -c ocean-continent.tgz | tar xvf -
