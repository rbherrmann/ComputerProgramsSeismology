Velocity model download and installation

The downloadable file models.tgz contains some velocity models that are used for creating synthetics. To unpack these for use in making velocity models, do the following:

  • Create a directory named GREEN which will contain the Green's functions for the individual velocity models
  • Note the absolute path of this directory. For example this might be/backup/rbh/PROGRAMS.310t/GREEN
  • Set the environment parameter GREENDIR in the .bashrc file if you use the bash shell. [If you use another shell, set up something similar]. The line to be added to .bashrc for the parameter above is
    export GREENDIR=/backup/rbh/PROGRAMS.310t/GREEN
  • Refresh the environment
     source ~/.bashrc

    Note this will be done automatically when you login again.
  • Install the models: 
    cd $GREENDIR 
gunzip -c models.tgz | tar xvf -

    This will create the subdirectory Models with the model files

Velocity models

The following velocity models are included in this distribution:

  • CUS.mod - model for North America craton
  • WUS.mod - model for North America with lower velocities in the upper crust
  • CIA.mod - model for the central Italian Apennines
  • KOREA.mod - model for the southern Korean peninsula
  • tak135sph.mod - continental AK135 model for teleseisms
